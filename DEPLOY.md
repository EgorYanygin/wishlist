# Deploy guide - GitHub Pages

## GitHub Pages

1. Открой репозиторий на GitHub
2. Settings → Pages
3. Source: ветка `main`, папка `/ (root)`
4. Save
5. Сайт доступен по `https://USERNAME.github.io/REPO`

> `.nojekyll` уже добавлен — Jekyll отключён.

## Vercel

1. Импортируй GitHub-репо в Vercel
2. Framework preset: Other/static site
3. Deploy — URL готов.

## Netlify

GitHub integration → выбери репо → Deploy.
